Related Doc: package markov

object MarkovGraph

Objet contenant les fonctions pour l'étude de graphes de Markov.

Ce module implémente : - Les calculs matriciels (multiplication, puissance, différence) - L'algorithme de Tarjan pour les composantes fortement connexes - L'extraction de sous-matrices et le calcul de distributions stationnaires
